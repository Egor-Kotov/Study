def{объявляем функцию} plus{название функции}(x, y){параметры которые мы передаем для выполнения функции}.
def plus(x, y):
{после двоеточия идёт код который выполняется с входными данными}
print(x + y)

В данном случаи функция будет каждый раз печатать сумму двух входящих цифр.
Для того что сохранить результаты функции используется оператор return.
Функции используются для разгрузки кода от повторяющемся команд. Внутри функции можно 
использовать иные операторы и функции. Переменные созданные в функции остаются, только 
внутри функции. Их невозможно вызвать в другой части кода.

Параметры:
Возможно установить параметры по умолчанию. Он будет автоматически проставляется, если мы его пропустим в скобках 
def gog(g, o, v = 5). В данном случае при пропуске значения в скобках, автоматом подставится 5 (gog(5,6)).
Если не знаем, в каком порядке идут наши параметры, можем прописать их при запуске функции (gog(v=8,o=6,g=5)).
Когда можно вести любое количество параметров используем (gog(*args))
