Тип данных - словарь. Это неупорядоченный список объектов в виде {key : value, key : value}. 
Обычно используем этот способ хранения данных, когда у нас есть данные связанные друг с 
другом(пример - телефонная книга, номер связан с именем и наоборот). 

Поиск в словарях происходит, не по индексу(тк данные там не упорядочены), 
а по ключу print(tel_b["Egor"]). Добавление нового значения в словарь, 
также происходит по ключу tel_b["Yn"] = 1234. Редактирование - при присутствии такого ключа в словаре 
его значение заменяется на указанное. Для удаления пары key-value используется keyword - del tel_b["Yn"]. 
Для очищения всего словаря используем метод clear -  tel_b.clear(), получим пустой словарь {}. 
Для проверки наличия данных с таким ключом используем in - "Yn" in tel_b (False или True).

Для вывода всех keys в словаре используется метод tel_b.keys() для вывода всех values tel_b.values(). 
Для преобразования всего этого добра в лист используется метод list(tel_b.keys()) - лист ключей, 
если использовать лист на словаре сразу получим просто лист ключей  - list(tele_b)
